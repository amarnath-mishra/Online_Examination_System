--------For Authentication and Authorisation--------

For student:


Show Two Options--
1.Registration(signup)
2.Login (signin)

if Student already not have an account 			//signup process
{
Student Enter its credentials in Registration Forms--

	a:name
	b:Username
	c:password
	d:age
	e:gender
	......etc.
	
	Submit.
   if all necessary parametres filled 
	{
	Show Successful message.
	}
   else
	{
	show error messages for that box.
	}
	
Else							//Signin process
{
 User Enters his Login And password 
  if (User-name and password is Correct)
	{
	Show Successful Login message.
	}
   else
	{
	show error message for LoginID/Password.
	and ask to try again.
	}
}
